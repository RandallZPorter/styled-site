Aside

I made my own programming language a few weeks ago. I named it "Bur." Here are some features of this language:

  1. It is interpreted by an interpreter I wrote in Java
  2. The only explicitly defined math is devision
  3. Dividing by zero will randomly increment or decrement the top of the stack
  4. Loops can be simulated with recursive functions
  5. It is slow, even compared to PowerShell and Python
  6. Whitespace is ignored
